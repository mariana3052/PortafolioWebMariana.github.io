Proyecto Comunidad de Respuestas

Se necesita realizar un proyecto que funcione como una comunidad de preguntas y respuestas estilo Stack Overflow.

Resolución del problema

Se realizo la página web donde en el menú principal podrá ser consultada sin necesidad de estar registrados.
Solo los usuarios registrados puede publicar una pregunta y contestar.

Un usuario, al iniciar una pregunta, puede escribir un título, una descripción del problema y subir una imagen.
La pregunta deberá ser catalogada como: MySQL, SQL Server, C#, C++, Java Script o NodeJS

La pregunta se puede editar o eliminar. Los usuarios pueden contestar la pregunta.
La página principal tiene una sección donde pueden buscar preguntas por su Id.
La página principal al logearse tiene una sección llamada MIS PREGUNTAS donde se listan las preguntas que el usuario ha publicado.
La página principal al logearse tiene una sección llamada MIS RESPUESTAS donde se listan las respuestas que el usuario ha publicado.
